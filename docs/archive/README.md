# Archived documentation

Markdown files in this folder are **historical snapshots** (phase reports, one-off analyses, completion banners). They may describe features or timelines that no longer match the codebase.

**Current sources of truth**

- Root `README.md` — how to run and test the app.
- `DEPLOYMENT.md` and `docs/OBSERVABILITY-AND-DEPLOYS.md` — deploys and operations.
- `ENV-VARS-QUICK-REFERENCE.md` — configuration.
- `PROJECT-ENDPOINT-MATRIX.md` — API surface (verify against code when in doubt).

When updating behaviour, prefer changing code and the docs above rather than editing archived files.
