# MT4 Price Bridge

Fresh local bridge for live ERP metal prices from your connected Equiti MT4 account.

The bridge is an MT4 Expert Advisor (`EquitiMetalPriceBridge.mq4`). Keep MT4 logged in to account `3140019` on `EquitiSecurities-Live 3`, attach the EA to any chart, and it will send Gold, Silver, and Platinum quotes to the backend every second.

## Flow

```text
Equiti MT4 account 3140019
  -> MT4 EA reads Market Watch bid/ask ticks
  -> POST /api/erp-accounting/metal-rates/bridge
  -> backend saves latest tenant metal rates
  -> Socket.IO broadcasts metal-rates:update
  -> React tenant top bar changes in real time
```

## Backend setup

Set a shared token in `backend/.env`:

```text
METAL_RATES_BRIDGE_TOKEN=change-this-long-random-token
```

For local development, the backend URL is:

```text
http://localhost:5000/api/erp-accounting/metal-rates/bridge
```

For production (Loopc / MG), use:

```text
https://api.loopcstrategies.com/api/erp-accounting/metal-rates/bridge
```

Railway env var (already required on production API):

```text
METAL_RATES_BRIDGE_TOKEN=<long random secret shared with MT4 EA>
```

Print MG-ready EA inputs from Railway (includes token + saves `mg-production.local.set`):

```powershell
powershell -File tools/mt4-price-bridge/scripts/print-mg-bridge-settings.ps1
```

Verify token + bridge POST:

```powershell
powershell -File tools/mt4-price-bridge/scripts/verify-bridge.ps1
```

Restart the backend after changing `.env`.

## MT4 setup

1. Open MT4 and confirm the title shows account `3140019` on `EquitiSecurities-Live 3`.
2. In MT4, open **File > Open Data Folder**.
3. Copy `EquitiMetalPriceBridge.mq4` into `MQL4/Experts/`.
4. Restart MT4, or right-click **Navigator > Expert Advisors > Refresh**.
5. Open **Tools > Options > Expert Advisors**.
6. Enable **Allow WebRequest for listed URL**.
7. Add the backend origin, not the full route:

```text
http://localhost:5000
```

For production (MG / CG / Loopc), add:

```text
https://api.loopcstrategies.com
```

For production MG, set inputs:

```text
BridgeUrl=https://api.loopcstrategies.com/api/erp-accounting/metal-rates/bridge
BridgeToken=<same value as Railway METAL_RATES_BRIDGE_TOKEN>
Tenant=mg
GoldSymbol=XAUUSD.pr
SilverSymbol=XAGUSD.pr
PlatinumSymbol=XPTUSD.pr
PostEverySeconds=1
```

Or load the preset generated by `scripts/print-mg-bridge-settings.ps1` (`mg-production.local.set`).

8. Drag **EquitiMetalPriceBridge** onto any chart and apply the inputs above (or load the `.set` preset).

9. Turn on **AutoTrading** (toolbar button must be green).

## Symbol names

If the EA shows `missing tick`, your broker uses different symbol names. In MT4 Market Watch, right-click and choose **Show All**, then find the exact names for:

- Gold: Equiti repo default `XAUUSD.pr` (often `XAUUSD` on other brokers)
- Silver: default `XAGUSD.pr` (often `XAGUSD`)
- Platinum: default `XPTUSD.pr` (often `XPTUSD`)

Some brokers add suffixes, such as `XAUUSDm`. Put the exact names into the EA inputs.

### Equiti: `XAUUSD` vs `XAUUSD.pr`

Equiti often lists **both** `XAUUSD` and `XAUUSD.pr`. The EA reads whatever you type in `GoldSymbol` / `SilverSymbol` / `PlatinumSymbol`—it must match a **live** Market Watch row character-for-character.

- **Either** symbol works if it ticks and the bridge posts successfully.
- Prefer **one family for all three metals** (all `.pr` or all plain) to avoid confusion.
- If you want prices aligned with the **`.pr`** feed (common on Equiti), the EA defaults are already **`XAUUSD.pr`**, **`XAGUSD.pr`**, **`XPTUSD.pr`**. For brokers without `.pr`, set plain symbols in the EA inputs.

See also [`docs/MT4_METAL_PRICE_BRIDGE.md`](../../docs/MT4_METAL_PRICE_BRIDGE.md).

## Expected result

The selected tenant top bar should update every second. The top bar displays MT4's original **USD/TOZ** tick values, while the backend also stores converted USD/gram values for ERP calculations.

## Troubleshooting

- `401 Invalid metal rates bridge token`: token in EA does not match `METAL_RATES_BRIDGE_TOKEN`.
- `503 Metal rates bridge is not configured`: backend `.env` is missing `METAL_RATES_BRIDGE_TOKEN`, or backend was not restarted.
- `WebRequest failed 4014`: MT4 WebRequest URL is not allowed in **Tools > Options > Expert Advisors**.
- `missing tick`: symbol is not visible or the symbol name is wrong.
- Top bar still says `waiting MT4`: the EA is not posting successfully, the EA `Tenant` input does not match your logged-in company, or your browser is logged into a different tenant.
